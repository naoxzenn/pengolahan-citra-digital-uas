# Deteksi Sampah Menggunakan Pengolahan Citra Digital

Proyek Ujian Akhir Semester (UAS) Mata Kuliah **Pengolahan Citra Digital**.

Dapat diakses disini : https://jupyter.org/try-jupyter/lab/index.html?path=notebooks%2FDeteksi_Sampah.ipynb
## Deskripsi

Program ini mendeteksi objek sampah pada citra menggunakan teknik pengolahan citra digital berbasis Python dengan memanfaatkan OpenCV, NumPy, dan Matplotlib.

## Fitur

- Membaca citra
- Preprocessing
- Gaussian Blur
- Segmentasi
- Operasi Morfologi
- Contour Detection
- Bounding Box
- Visualisasi hasil deteksi

## Library

- Python
- OpenCV
- NumPy
- Matplotlib

## Cara Menjalankan

1. Install library yang dibutuhkan.
2. Buka file `Deteksi_Sampah.ipynb`.
3. Jalankan seluruh cell secara berurutan.
4. Ganti gambar pada variabel `IMAGE_PATH` jika ingin menguji citra lain.
